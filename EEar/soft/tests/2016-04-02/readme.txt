1. Отформатировать карточку в FAT32
2. Вставить карточку в плату
3. Прошить плату и запустить тест

Результат:
если горит D4 - ошибка инициализации SD_Init()
если горит D3 - нет SD-карты
если горит D2 - ошибка file i/o
если загорелся D1, то в течение следующей минуты будет идти запись с микрофона в файл out.raw, кот. будет создан.
Потом спустя минуту D1 гаснет.
Запись ведется в формате PCM (моно, 16000 Гц, 16-бит little-endian)
