Два теста для тестирования приема/передачи через UART.

25mhz_usart3_pb11_pb10.bin  - ТОЛЬКО(!) для твоей платы
8mhz_usart3_pb11_pb10.bin   - ТОЛЬКО(!) для DISCOVERY BOARD

Оба теста используют:
- порт USART3
- пин TX: PB10 (от платы к терминалу)
- пин RX: PB11 (от терминала к плате)
- скорость: 9600 (8N1)

Как тестировать: 
- подключить RX/TX к терминалу на компе
- печатать символы на терминале
- с каждым принятым символом будут кратковременно загораться все 4 лампочки LED на 100 мсек
- каждый принятый символ будет тут же "эхом" отправляться назад через этот же порт на терминал

Внимание!
Тест не предусматривает отправки/приема крупных пакетов данных. Отправлять по одному символу с клавиатуры!
