1. Отформатировать карточку в FAT32
2. Создать в корне файл test.txt и записать туда строчку типа "Hello world!"
3. Вставить карточку в плату
4. Прошить плату и запустить тест

Результат:
если горит D4 - ошибка инициализации SD_Init()
если горит D3 - нет SD-карты
если горит D2 - ошибка file i/o
если горит D1 (с проблымом в 600 мсек) - все ок! В корне будет создан файл out.txt, в котором 100 раз дублируется
текст из файла test.txt
