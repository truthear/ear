Тесты:
1) 25MHz_HSE.bin - проверяет правильность установки опорной частоты кварца 25 МГц (HSE)
2) 25MHz_ticks.bin - проверяет, действительно ли частота CPU 168 МГц (методом считывания счетчика тиков CPU)
3) 25MHz_LSI.bin - для мигания использует RTC (внутренний осциллятор)
4) 25MHz_LSE.bin - для мигания использует RTC (внешний осциллятор 32768 Гц)

Ожидаемые результаты:
Все тесты используют 4 *РАЗНЫЕ* лампочки (т.е. для каждого теста будет мигать своя лампочка!!!).
Периодичность мигания лампочек для всех тестов должна быть 1 сек (т.е. 1 сек горит, 1 сек не горит и т.д...),
если период не 1 сек (что явно должно быть видно), или вообще не горит ничего, значит проблема...
Должны работать все 4 теста!
