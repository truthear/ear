   - If the Erase operation is PASSED then LED D1 ON else the LED D4 is ON and LED D1 
     is OFF 
   - If the Single Block Write/Read operation is PASSED then LED D2 ON else the LED D4 
     is ON and LED D2 is OFF
   - If the Multi Blocks Write/Read operation is PASSED then LED D3 ON else the LED D4 
     is ON and LED D3 is OFF
   - Any SD Card operation including the SD Initialization error is signaled by
